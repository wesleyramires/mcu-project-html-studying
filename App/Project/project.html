O começo de um projeto traz consigo grandes desafios. Esse projeto foi criado, afim de explorar novos campos de desenvolvimento para web. Tenho como objetivo o desenvolver um site com conteúdos cinematográficos da Marvel, ao qual, é um dos universos das HQ preferidos. Nesse primeiro processo, estou desenvolvendo o site, utilizando somente HTML 5 e CSS 3 para abranger os conhecimentos da linguagem. Na segunda etapa, começarei a incluir no desenvolvimento do site o javascript e posteriormente, usarei frameworks como bootstrap. Todo o projeto visa o estudo desse meio do mundo developer.
